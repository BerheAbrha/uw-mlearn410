Linear Regression (Ordinary Least Squares)

  • A technique for supervised learning that is based on strong statistical assumptions about the underlying data.
  • It can be used when the assumptions are not met, but then it needs to be graded based on performance on the test data.
  • P-values, F-test statistics, etc…depend on the assumptions being true.
